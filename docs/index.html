Bem-vindo ao Projeto HealthCodec

Healthcodec é um formato compacto e eficiente para armazenar dado em saúde registrado em conformidade com objetos do modelo de referência do openEHR.

Google faz uso de Protocol Buffers e FlatBuffers, Netflix usa Hollow, Github usa BERT, Redis usa MessagePack e Kryo serializa grafos de objetos em Java, dentre outras opções. Também pode ser citado formatos como ORC dentre outras opções. Healthcode é uma estratégia específica para informação em saúde, e mais eficiente.
